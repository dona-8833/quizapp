# Quiz App

This is a simple quiz application that allows users to take quizzes and see their scores.

## Features

- Multiple choice questions
- Score tracking
- User-friendly interface

## Installation

1. Clone the repository:
    ```
    git clone https://github.com/yourusername/quizapp.git
    ```
2. Navigate to the project directory:
    ```
    cd quizapp
    ```
3. Install the dependencies:
    ```
    npm install
    ```

## Usage

1. Start the application:
    ```
    npm start
    ```
2. Open your browser and go to `http://localhost:3000` to access the quiz app.
